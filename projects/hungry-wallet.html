Hungry Wallet

HungryWallet is a brand new android mobile application that presents a clever way of managing the amount you spend at restaurants. It is straightforward in making sure you are spending within your means, and offers a simple and intuitive user experience. The user interface is clean and beautiful with colors that evoke feelings of appetite and desire.

It is created as a part of UCSD Software Engineering class (CSE110), where multiple teams of enrolled students are asked to developed a restaurant themed app, using any technology and implementation, solving any restaurant related problem.

It is developed using the Eclipse Android Developer Tools (ADT) plugin that runs on the Java programming language. It utilizes the parse cloud platform to host its backend, ensuring reliable and fast data processing. The search functionality, utilizes the yelp and google places API, to ensure reliable and accurate information given to the user.

On the other hand, the charts and graphs, used a derivation of open source libraries, to complement the UI design principles chosen.
